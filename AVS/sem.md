# Архитектура выч. систем

## Третий семинар

Рассказывают как конпелять.

Чтобы сгенерировать ассемблерный код нужно указать флаг "-S" и ещё можно указать "-fverbose-asm" для создания комментариев. После этого появится файл с суффиксом ".s".

Если переменная объявлена в функции она хранится в стэке, для работы с памятью в стэке есть rsp -- в нём хранится последний помещенный в стэк элемент. Мы выделяем в стэке 16 байт:

```asm
subq $16, %rsp #,
```

Основной регистр для работы со стэком это rsp, а вспомогательный --- rbp.

```asm
movl  $-100, -12(%rbp) # x,
movl  $4, -8(%rbp)  # y,
movl $0, -4(%rbp) # z,
```

Здесь обращение идёт к rbp со смещением на -12, переменные хранятся со смещеением. Переменные инициалированы, поэтому мы запихиваем в них константы.

$ это константа, а без доллара --- обращение к какой-то ячейки памяти.

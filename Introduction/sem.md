- Изобретение:
  - Ёмкость для хранения и выдачи пастообразных продуктов питания, используемая в вендинговых аппаратах

Изобретение относится к таре для вендинговых аппаратов, реализующих пастообразные продукты.

Ёмкость для хранения и выдачи пастообразных продуктов состоит из цилиндрического сосуда (1) с двумя открытыми торцами. Внутрь сосуда вставляется поршень (2) с уплотнительными кольцами (4). С одной стороны поршня выполнен выступ (шток, 8) для соединения с приводом дозатора.

Технология изготовления и применения:

   После установки поршня торец сосуда со стороны штока укупоривается вкладышем (5) и закрывается винтовой крышкой (6).

   Через второй торец ёмкость наполняется продуктом, после чего также укупоривается вкладышем и закрывается крышкой.

   Заполненная ёмкость подвергается термической обработке.

   В вендинговом аппарате крышки снимаются, вкладыши вскрываются, а шток поршня подключается к дозатору. При движении поршня продукт выдавливается наружу.

Технические результаты:

   Герметичность на всех этапах (от фасовки до выдачи) предотвращает контакт продукта с воздухом, увеличивая срок хранения.

Термостойкость материалов позволяет проводить термическую обработку уже наполненной ёмкости.

 Конструкция с поршнем обеспечивает дозированную выдачу продукта без попадания воздуха внутрь.

Уровень техники:
Известные решения (бутылки, контейнеры для снеков) не решают проблему хранения и выдачи пастообразных продуктов: они негерметичны, не позволяют извлекать продукт без доступа воздуха и не выдерживают термообработку. Наиболее близкий аналог (двухзонная ёмкость для снеков) также имеет эти недостатки.

Сущность конструкции:
Ёмкость (Фиг. 1) включает: цилиндрический корпус (1), поршень (2) с канавками (3) и кольцами (4), шток (8), два герметизирующих вкладыша (5) и две винтовые крышки (6). Поршень перекрывает сечение сосуда, а вкладыши обеспечивают полную герметизацию при хранении и термообработке.

- Формула изобретения:
  - Ёмкость для хранения и выдачи пастообразных продуктов питания, используемая в вендинговых аппаратах, состоящая из цилиндрического сосуда с открытыми обеими торцевыми сторонами, во внутреннюю часть сосуда вставляется поршень с уплотнительными кольцами, с одной стороны поршень ровный, со второй выполнен выступ цилиндрической формы, который предназначен для соединения поршня со штоком дозатора, после установки поршня эта часть цилиндрического сосуда укупоривается вкладышем для индукционного или клеевого пломбирования и закрывается винтовой крышкой твист офф, с другой стороны цилиндрический сосуд после наполнения готовым продуктом укупоривается вкладышем для индукционного или клеевого пломбирования и закрывается винтовой крышкой твист офф.
